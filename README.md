# Python商品评论分析系统简介

## 核心功能
- 登录注册
- 商品评论数据总览
- 商品评分分析
- 年月时间分析
- 订单评论数据分析
- NLP评论情感分析
- 商品评论词云图
- LDA评论主题分析

## 适用场景
- 商品评论数据可视化分析
- 电商平台用户反馈研究
- 产品改进与市场调研
- 顾客满意度评估

## 技术特点
- 使用 Django 框架搭建，保障系统稳定性和扩展性
- MySQL 数据库存储，确保数据安全与高效查询
- Vue 前端技术实现动态交互，提升用户体验
